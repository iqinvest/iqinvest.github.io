faq

  • What's iq invest?

    Iq invest is the small company consisting of blockchain developers and economists. Our goal is to develop and share the blockchain wisdom by mastering the existing market and creating the new ones.

  • What's token?

    Token is an ethereum-based share. Ethereum is the first world-wide money that allows applications to directly use this money. Tokens are stored in the ethereum network and regulated by computer code, unlike ordinary shares, which are controlled by an obsolete national economy. By owning a token, you own an ethereum-based company.

  • What's this website for?

    This is a market where anyone can buy and sell iq token.

  • Why do I need metamask?

    Metamask is an ethereum wallet. You need it in order to connect to the ethereum network and keep your accounts. Follow the instructions to install and use your wallet.

  • Which network should I use?

    Iq token deployed on main network and ropsten test network. You can use both of it. Tokens and ethereum have a market value only in the main network. You can use the ropsten test network to understand how the exchange works. The token and exchange in the test network have exactly the same functionality, but the ropsten ethereum is free. Contact us if you want to use any other (not ropsten) test network.

  • How is the token price determined?

    The token price depends on the number of tokens on the exchange. We use a simple linear function, because of the easier calculation and therefore lower trasaction fee.

    p = 1 - v/5000000, where

    p - price, eth/token

    v - number of tokens on the exchange

    This way the law of supply and demand is emulated without an order book, which makes it very easy to trade tokens.
