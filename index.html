What's token?

Token is an ethereum-based share. Ethereum is the first world-wide money that allows applications to directly use this money. Tokens are stored in the ethereum network and regulated by computer code, unlike ordinary shares, which are controlled by an obsolete national economy. By owning a token, you own a part of an ethereum-based company.

Which network should I use?

Iq token is deployed in the mainnet and ropsten testnet. You can use both. Tokens and ethereum have a market value only in the mainnet. You can use the ropsten test network to understand how the exchange works. The token and exchange in the test network have exactly the same functionality, but the ropsten ethereum is free. Contact us if you want to use any other (not ropsten) test network.

How is the token price determined?

The token price depends on the number of tokens in the exchange. We use a simple linear function, because of the easier calculation and therefore lower trasaction fee.

p = 1 - v/5000000, where

p - price, eth/token

v - number of tokens in the exchange

This way the law of supply and demand is emulated without an order book, which makes it very easy to trade tokens.
